# Finance Skill Interface

This directory is interface-only. It describes public safety boundaries for finance, stock, investment, company analysis, and technical-indicator skill packs.

Concrete finance skills should be maintained as private packs when they depend on proprietary methods, paid data sources, internal assumptions, or regulated workflows.

## This Public Repository Does Not Include

- Stock screening rules
- Buy, sell, or hold decision logic
- Technical indicator combination strategies
- Valuation model weights
- Stock pools
- Portfolio construction strategies
- Paid data source integration logic
- Proprietary research templates

## Public Interface Scope

This package only defines generic safety principles and output structure. It does not provide personalized investment advice and does not guarantee returns.
