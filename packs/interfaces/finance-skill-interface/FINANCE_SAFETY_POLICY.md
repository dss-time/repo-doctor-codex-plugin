# Finance Safety Policy

Finance, stock, investment, company analysis, and technical-indicator skills should be private when they contain implementation logic.

## Required Safety Principles

- Always state the data date.
- Always state data sources.
- Separate facts, inferences, and opinions.
- Report risk factors.
- Do not provide personalized investment advice.
- Do not guarantee returns.
- Require human review before decisions.
- Mark missing, stale, or unverifiable data.

## Public Repository Exclusions

The public repository must not include:

- Stock screening rules
- Buy, sell, or hold logic
- Technical indicator combination strategies
- Valuation model weights
- Stock pools
- Portfolio strategies
- Paid data source integration logic
- Private research workflows

## Private Pack Guidance

Private finance packs should define their own governance, data licensing, compliance review, and approval process before use.
