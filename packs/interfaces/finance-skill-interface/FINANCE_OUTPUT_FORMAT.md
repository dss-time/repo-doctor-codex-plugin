# Finance Output Format Interface

This is a generic interface only. It is not a strategy.

## Recommended Sections

1. Scope and user question
2. Data date
3. Data sources
4. Facts
5. Inferences
6. Opinions or scenarios
7. Key uncertainties
8. Risk factors
9. Human review checklist
10. Non-advice disclaimer

## Required Distinctions

| Section | Requirement |
|---|---|
| Facts | Must cite source and date. |
| Inferences | Must explain assumptions. |
| Opinions | Must be labeled as opinion or scenario. |
| Risks | Must include material uncertainty and data limitations. |
| Review | Must identify what a human should verify. |

## Prohibited Content in Public Outputs

Do not include concrete screening formulas, trading signals, valuation weights, portfolio rules, paid data connectors, or personalized recommendations.
