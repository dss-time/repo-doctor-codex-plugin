# Document Data Doctor Pack

Document Data Doctor contains public-safe office workflow skills.

The skills are generic. They do not include legal conclusions, internal templates, financial advice, or customer-specific examples.

Every review should cite evidence such as page numbers, paragraph numbers, sheet names, columns, fields, or source sections when available.

The former basic `report-writer` now has one enhanced canonical implementation in the independent `productivity-toolkit` pack. Its public invocation name remains `report-writer`.
