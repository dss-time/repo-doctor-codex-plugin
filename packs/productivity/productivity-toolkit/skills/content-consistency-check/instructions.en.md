# Content Consistency Check

Read `../../references/productivity-evidence-and-file-safety.md` and `references/consistency-dimensions.md`.

1. Confirm the artifacts, comparison scope, authoritative sources if any, effective dates, and dimensions to compare.
2. Inventory actual format capabilities. Extract only what available tools can support from Markdown, text, documents, spreadsheets, PDFs, or presentations; report inaccessible files, sheets, slides, pages, formulas, or visuals.
3. Create a normalized comparison layer for terms, figures, dates, units, identities, versions, links, and conclusions while preserving each original value and location.
4. Classify each difference as a true conflict, formatting-only difference, unit conversion, rounding difference, time-basis difference, version/effective-date difference, or unknown.
5. Report both sides, context, impact, severity, confidence, and proposed authority. Do not silently choose the correct value; select one only when authoritative evidence establishes it.
6. Summarize systemic patterns and a reconciliation sequence. Keep the source artifacts read-only.

Use `document-review` when the request is primarily structure, logic, grammar, and completeness within one document.
