# 通用办公与研究生产力工具包

Productivity Toolkit 是独立、公开安全的通用办公与研究 Pack，不包含组织专属规则、私有模板、投资策略、财务估值、选股或交易建议。

包含：

- `report-writer`
- `research-brief`
- `spreadsheet-data-cleaning`
- `document-review`
- `pdf-review`
- `meeting-notes-to-actions`
- `presentation-outline`
- `content-consistency-check`

所有 Skill 默认保护原始材料，区分证据与推断，跟随用户语言；缺少文件、浏览器、搜索、OCR、渲染或执行工具时必须明确降级。
