# Skill Maintainer Pack

Skill Maintainer contains two public-safe workflows for repository maintainers:

- `skill-authoring` creates or updates one well-scoped AI Skill through the repository's existing scaffold and build system.
- `skill-quality-audit` performs a strictly read-only, evidence-backed pre-release audit of a Skill, Pack, plugin, or Skills repository.

The pack separates deterministic validation from model judgment. It contains no business-domain workflow, private implementation, customer information, credential, or restricted investment logic.
