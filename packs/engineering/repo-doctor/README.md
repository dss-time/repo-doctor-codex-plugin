# Repo Doctor Pack

Repo Doctor is a public-safe engineering pack for repository-level AI-assisted work.

It includes:

- `repo-onboarding`
- `project-health-check`
- `safe-code-review`
- `change-impact-analysis`
- `safe-fix-implementation`

The pack defaults to read-first behavior. Editing is only appropriate in `safe-fix-implementation`, and only after a clear diagnosis and validation plan.
